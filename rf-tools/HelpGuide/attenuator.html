Attenuator

The attenuator design tool is separated into four (4) distinct attenuator architectures: pi, tee, bridged-tee, and reflective. The user may choose among these architectures by clicking on the related tab docked to the left of the window. Going down the tabs the user can see there is the pi, tee, bridged-tee and reflective attenuators. Clicking one of these tabs will activate that specific design tool. The image to the left depicts the pi attenuator design tool.

On the top right of the window is a help button that takes the user to this design guide for help relating to certain controls. Below this is an image of the attenuator schematic with reference designators to indicate the value associated with each resistor.

Under the Synthesis label is the attenuation (Atten)input, the characteristic/system impedance (Z0) input, and the resistor tolerance (Tol) input. The attenuation value should be entered as a positive number; negative numbers mean a gain which is not feasible in a purely linear circuit. Along with the system impedance, the attenuation value is used to determine the ideal resistor values. From here the tolerance of the resistor is used to determine the standard resistor values required. Once the calculations are completed the boxes under the Evaluate label are filled with the appropriate values.

Under the Evalute label is the user input for the resistances used in the existing attenuator. These resistor values are used to approximate the system impedance and attenuation the for which the attenuator was designed.

Above the Synthesis and Evaluate buttons is checkbox to generate a report. When checked, a report is generated after the user clicks either Synthesis or Evaluate. This report contains the following information,

  • Resistor Values
  • Approximate Nominal Attenuation
  • Approximate Nominal Return Loss
  • Attenuation Variation
  • Estimated Maximum Power

Power Handling

The power handling capability of the presented attenuators depend on the selected resistor rating. The basic idea is that the power dissipation is spread among the resistors in the circuit, and the total amount of power dissipated is related to the attenuation value. In essence this means that lower valued attenuators can handle higher power levels.

In order to further increase the power handling of the circuit there are a couple paths to follow:

  1. Increase Package Size
  2. Material Change

Increasing package size will yield a higher power rating at the expense of board real-estate and wideband performance. For example going from 0402 resistors to 0603 might double the total amount of continuous sustainable power level. For many applications this may be enough.

However, this is not always acceptable. Certain circumstances will require 50 W (or higher) handling at 50% de-rating. Designing an attenuator for continuous 100 W operation cannot be done by simply increasing package size. Situations such as the one presented would warrant the use higher power materials at the expense of broadband performance.

Many materials exist for resistors,

  1. Thin Film
  2. Thick Film
  3. Metal Film
  4. Wirewound
  5. Cemented Wirewound

The generated report displays power handling assuming an 0402 1/16 W resistor is being used. The way to convert this number into the designed application is by the following equation

power + 10 * log(16 * rating)
