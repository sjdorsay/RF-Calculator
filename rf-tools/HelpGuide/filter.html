Filter

Overview

The filter design tool is separated into configuration and implementation tabs. When the configuration tab (the gear icon) has been selected the user can input their desired filter response. These values are then stored internally to be used for all the filter implementation tabs: the prototype only needs to be specified once. When the required information has been added, select the filter implementation by clicking on the related tab. From here the user should fill in any associated data before clicking "Synthesize".

Configuration

In the configuration tab the user can select the filter type: LPF, HPF, BP, or BSF; the filter order: 1 - 10; the filter architecture: butterworth, chebyshev, or linear phase; and the first element: series, or shunt. Once these parameters have been selected the user can enter the system impedance, ripple, bandwidth, and center/cutoff frequency. The tool calculates the prototype values assuming 3 dB cutoff frequency for all filter architectures. This information is important to takenote of since the natural cutoff frequency of the chebyshev filter is not the same as the 3 dB cutoff frequency.

Lumped Element

The lumped element filter is the realization of the filter prototype as inductors and capacitors. In the Synthesis section is a drop-down box that allows the user to select between ideal component values and standardized component values. This tool also allows for setting minimum component values.
