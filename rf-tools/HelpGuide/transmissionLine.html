Transmission Line

The transmission line design tool is separated into five (5) distinct geometries: microstrip, stripline, Coplanar Waveguide, Grounded Coplanar Waveguide, and Substrate Integrated Waveguide (SIW). The user may choose among these geometries by clicking on the related tab docked to the left of the window. Clicking one of these tabs will activate that specific design tool. The image to the left depicts the microstrip design tool.

On the top right of the window is a help button that takes the user to this design guide for help relating to certain tools and controls. Below this is an image of the selected trace geometry and substrate information that the user can fill in to match their needs.

Under the Synthesis label is the system impedance (Z0) input, and the electrical length/phase input. The system impedance and the phase should be entered as positive numbers; negative impedance will be assumed to 50 Ohms and may yield incorrect output values. Along with the system impedance, the attenuation value is used to determine the ideal resistor values. From here the tolerance of the resistor is used to determine the standard resistor values required. Once the calculations are completed the boxes under the Evaluate label are filled with the appropriate values.

Under the Evalute label is the user input for the resistances used in the existing attenuator. These resistor values are used to approximate the system impedance and attenuation the for which the attenuator was designed.
